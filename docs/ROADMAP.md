# Roadmap

This roadmap is not a promise. It is a planning document for the next development steps.

## v1.35 — Quick Search Foundation

- Quick exact nickname search.
- Compact search result strip.
- Add searched nickname to watched list.
- Spanish language option.
- Region selector for map alerts.
- Search logic isolated from UI.

## v1.40 — Suggestions

- Autocomplete for online nicknames.
- Suggestions from the current DDNet server list.
- Click suggestion to fill the search box.

## v1.50 — Search BOOM

- Stronger search foundation.
- Better favorite player/map flows.
- More polished quick actions.
- Preparation for player profile screens.

## v1.65 — Player Stats Skeleton

- Basic player profile screen.
- Nickname, total points, country and skin if a stable data source is available.
- Foundation for future analytics.

## Later

- Discord and Telegram notifications.
- Tray mode.
- Startup with Windows.
- Player progress timeline.
- Map and points analytics.
- Player comparison tools.
