# DDNetNW v1.20 Release Notes

DDNetNW v1.20 expands the app from a nickname watcher into a player and map activity watcher.

## Highlights

- New Players / Maps mode in the main menu.
- Watched maps are shown as folders.
- Opening a map shows active servers running that map.
- Map alerts can notify when a watched map reaches a selected online threshold.
- Map alert filters can limit notifications to server names such as GER or RUS.
- UI language and theme are now custom segmented switches, not default dropdowns.
- About page now explains data source, privacy and limitations.

## Suggested GitHub Release text

