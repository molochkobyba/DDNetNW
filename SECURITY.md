# Security Policy

## Supported versions

The latest public release is the supported version. Older experimental versions may not receive fixes.

## Reporting a vulnerability

If you find a security issue, please open a private report if available, or contact the repository owner.

## Sensitive data

Do not commit:

- Telegram bot tokens;
- Discord webhook URLs;
- passwords;
- private API keys;
- personal test settings.

DDNetNW is designed to store user settings locally and does not require a private backend for the current local notification features.
